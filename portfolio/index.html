My Reflections and Journey of Learning Flutter

Week 1 – Introduction to Flutter

In my first week learning Flutter, I started with understanding what Flutter is and the basics of its architecture. I installed the Flutter SDK and set up my environment for development.

I created my first project and built a simple app that displays "Hello Flutter!". It felt great to have a working app already! I also learned about the structure of Flutter applications, including the main function. It was a whirlwind of information, but I'm thrilled to be on this journey.

Week 2 - Creating a Hello World App

This week, I created my first complete app and really got hands-on with Flutter.

I learned how to use hot reload, which lets me see changes in my app right away. I also discovered how to style my app with themes and understand when to use stateless and stateful widgets. Adding external packages was useful, as it saves time and effort by building on existing code.

Overall, it was an exciting week, and I’m eager to dive deeper into Flutter!

Week 3 - Learning Dart Basics

In week three, I focused on learning the basics of Dart programming.

I explored how to comment my code for clarity and learned about the main() function that starts the app. I practiced declaring variables such as numbers, strings, and lists. Iterating through lists was new for me, and I found using common operator symbols for arithmetic and logic quite helpful.

I also got familiar with using classes to organize my code. Understanding how to work with external packages using the import statement was a key takeaway.

It was a foundational week, and I feel more confident in Dart!

Week 4 - Creating a Starter Project Template

This week, I learned how to set up a starter project for future apps.

I used the mkdir command to create organized folders for my project. I also built two important Dart files: main.dart for the app's entry point and home.dart for the Home widget.

This organizational skills boost will help me as I continue developing apps. I’m excited to see where this leads!

Week 5 - Understanding the Widget Tree

In week five, I explored the concept of the widget tree.

I learned that the widget tree consists of nested widgets and how it can get quite complex. To make my code cleaner, I found it helpful to create separate widget classes, which leads to easier management and improved app performance.

Focusing on keeping the widget tree shallow is something I want to practice. It’s been a key learning week!

Week 6 - Using Common Widgets

This week, I focused on using common widgets in Flutter.

I discovered the basic widgets that are essential for building mobile apps. Learning about different types of buttons and how to add images and icons was a highlight. I also explored how to modify widget appearances using BoxDecoration and validate data inputs using the Form widget.

Using MediaQuery to understand device orientation was particularly interesting. I'm excited about how these tools will enhance my app design!

Week 7 - Adding Animation to an App

In week seven, I dived into adding animations to improve user experience.

I learned how to use widgets like AnimatedContainer and AnimatedOpacity to create engaging animations. The idea of using multiple AnimationControllers was fascinating, as it allows for complex animations to be built simply.

Creating staggered animations was a fun challenge, and I’m eager to see how I can use these techniques in my apps!

Week 8 - Creating an App’s Navigation

This week, I focused on app navigation using the Navigator widget.

I learned how to manage multiple pages and create smooth transitions between them. The BottomNavigationBar was helpful for quick navigation, and I enjoyed implementing a Drawer for sliding menus.

Using animations for transitions added a nice touch to the user experience. I’m excited to apply these navigation strategies to my ongoing projects!
